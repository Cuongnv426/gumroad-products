## 🔧 TECHNICAL NOTES

### HTML to PDF Conversion Options:

**Option 1: Online (Easiest)**
- Go to html2pdf.app
- Upload ChatGPT-Prompts-Library-Premium.html
- Download PDF
- Time: 2 minutes

**Option 2: Command Line (Best)**
- Install: `sudo apt-get install wkhtmltopdf`
- Run: `wkhtmltopdf input.html output.pdf`
- Time: 5 minutes

**Option 3: Python (Flexible)**
- Install: `pip install weasyprint`
- Run: `weasyprint input.html output.pdf`
- Time: 5 minutes

### CSV to Google Sheets:
1. Go to sheets.google.com
2. Click "Create a new spreadsheet"
3. File > Open > Upload
4. Select CSV file
5. Done!
