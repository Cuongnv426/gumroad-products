STEP 1: INSTALL (5 minutes)
──────────────────────────
1. Make sure you're in: /root/clawd/content-bot
2. Install dependencies:
   
   pip install -r requirements.txt
   
   (This installs OpenAI, tweepy, mailchimp, etc.)

STEP 2: GET API KEYS (30 minutes)
────────────────────────────────
Minimum (just to test):
- OpenAI: https://platform.openai.com/account/api-keys
- Twitter: https://developer.twitter.com/

Recommended (to earn money):
- Also add Mailchimp: https://mailchimp.com

Full list in: API-KEYS-NEEDED.md

STEP 3: CONFIGURE (10 minutes)
──────────────────────────────
Set your API keys as environment variables:

   export OPENAI_API_KEY="sk-..."
   export TWITTER_API_KEY="..."
   export TWITTER_API_SECRET="..."
   export TWITTER_ACCESS_TOKEN="..."
   export TWITTER_ACCESS_TOKEN_SECRET="..."

Or create a .env file with these keys.

STEP 4: TEST (5 minutes)
───────────────────────
Test if everything works:

   python content-bot-main.py test

You should see:
   ✓ Configuration checked
   ✓ Platform connections verified

STEP 5: GENERATE CONTENT (2 minutes)
────────────────────────────────────
Generate your first batch:

   python content-bot-main.py generate 10

This creates 10 pieces of content. Check:
   - data/queue-2024-XX-XX.json

STEP 6: START BOT (1 minute)
────────────────────────────
Run the bot:

   python content-bot-main.py start

The bot will:
1. Start in the background
2. Begin posting at scheduled times
3. Track analytics in real-time
4. Generate new content daily

Press Ctrl+C to stop (or use screen/systemd for background)

TROUBLESHOOTING:
────────────────
Problem: "No such file or directory"
Solution: Make sure you're in /root/clawd/content-bot

Problem: "ModuleNotFoundError: No module named 'openai'"
Solution: pip install -r requirements.txt

Problem: "OPENAI_API_KEY not set"
Solution: export OPENAI_API_KEY="sk-..." OR create .env file

Problem: "Bot not posting"
Solution: Check logs: tail -f logs/bot.log

Problem: Database error
Solution: rm data/analytics.db (will recreate on next run)
