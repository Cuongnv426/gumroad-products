# 🤖 Content Bot - AI Tools & Productivity Niche

**Production-ready automated content generation and posting bot** that runs 24/7 to generate passive income from AI tool affiliate marketing.

## ⚡ Quick Start (5 minutes)

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Set OpenAI API key
export OPENAI_API_KEY="sk-..."

# 3. Generate content
python content-bot-main.py generate 10

# 4. Start 24/7 bot
python content-bot-main.py start
```

**That's it!** Bot will now:
- Generate 10-20 posts daily across all platforms
- Post automatically at optimal times
- Track analytics and revenue in real-time
- Build your email list
- Manage affiliate links
- Generate reports

## 🚀 Getting Started

### 1. Installation (5 min)
```bash
# Clone/download files
cd content-bot

# Install Python 3.8+
python --version

# Install dependencies
pip install -r requirements.txt

# Create directories
mkdir -p data logs
```

### 2. API Setup (30 min)
Get API keys for:
- **OpenAI** (required) - Content generation
- **Twitter** (required) - Tweet posting
- **Mailchimp** (recommended) - Email growth
- **Other platforms** (optional) - More channels

See `API-KEYS-NEEDED.md` for detailed instructions.

### 3. Configuration (10 min)
```bash
# Set API keys
export OPENAI_API_KEY="sk-..."
export TWITTER_API_KEY="..."

# Or create .env file
# See API-KEYS-NEEDED.md

# Edit config.py to customize
nano config.py
```

### 4. Test (5 min)
```bash
# Test content generation
python content-generator.py

# Test platform connections
python platform-integrations.py

# Test scheduler
python scheduler.py
```

### 5. Launch (1 min)
```bash
# Start bot
python content-bot-main.py start

# Or use interactive menu
python content-bot-main.py
```

## ⚙️ Configuration

Edit `config.py` to customize:

```python
# Change niche/keywords
NICHE = "AI Tools & Productivity"
CONTENT_KEYWORDS = [...]

# Adjust posting schedule
POSTING_SCHEDULE = {
    "twitter": {"times": ["09:00", "12:00", ...]}
}

# Add/remove affiliate links
AFFILIATE_LINKS = {...}

# Change timezone
TIMEZONE = "UTC"

# Enable dry-run (test without posting)
DRY_RUN = False

# Change check interval
CHECK_INTERVAL_MINUTES = 5
```

### Bot Not Running?
```bash

tail logs/bot.log

# Verify API keys

# Test manually
```

```bash

# Test each platform individually
```
