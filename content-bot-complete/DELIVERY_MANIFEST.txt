QUICK START
================================================================================

LOCAL TESTING:
  1. bash setup.sh
  2. cp .env.example .env
  3. nano .env (add OpenAI key)
  4. source venv/bin/activate
  5. python src/bot_main.py test

VPS DEPLOYMENT:
  1. Create VPS (Linode/DigitalOcean: $5-10/month)
  2. SSH: ssh root@your-ip
  3. apt update && apt install python3 python3-venv python3-pip git
  4. cd /opt && git clone repo && cd content-bot-complete
  5. bash setup.sh
  6. nano .env (add API keys)
  7. sudo cp deploy/content-bot.service /etc/systemd/system/
  8. sudo systemctl enable content-bot && sudo systemctl start content-bot
  9. sudo journalctl -u content-bot -f (watch it go!)

BASIC COMMANDS:
  • Generate content:  python src/bot_main.py generate 5
  • Post to Twitter:   python src/bot_main.py post twitter
  • Send email:        python src/bot_main.py email send "Subject"
  • Check status:      python src/bot_main.py status
  • View analytics:    python src/bot_main.py report
  • Run 24/7:          python src/bot_main.py start
