## CATEGORY 1: TASK MANAGEMENT & TODO (10 Prompts)

### 1. Smart Task Breakdown Prompt
**Prompt:**
"I have this task: [TASK]. Break it down into 5-7 specific, actionable subtasks that I can realistically complete today. For each subtask, estimate the time needed and difficulty level (easy/medium/hard). Format as a numbered checklist I can use immediately."

**Use Case:** Breaking large projects into manageable daily chunks
**Example Input:** "Write a quarterly business report"
**Example Output:** 
1. Gather Q4 sales data (30 min) - Easy
2. Compile customer feedback summary (45 min) - Medium
3. Create charts/visuals (60 min) - Medium
4. Write executive summary (45 min) - Hard
5. Proofread and format (30 min) - Easy

---

### 2. Priority Matrix Generator
**Prompt:**
"I have these 8 tasks to do this week: [LIST TASKS]. Use the Eisenhower Matrix (urgent/important) to categorize them. Show me which I should do first, which I should schedule, which I should delegate, and which I should eliminate. Explain your reasoning for each."

**Use Case:** Deciding what to focus on when overwhelmed
**Example Input:** "Email clients, fix website bug, plan vacation, call dentist, learn new software, attend meeting, social media post, grocery shopping"
**Example Output:** 
- DO FIRST (Urgent & Important): Fix website bug, Call dentist
- SCHEDULE (Important, Not Urgent): Plan vacation, Learn new software
- DELEGATE: Social media post, Email clients
- ELIMINATE: Grocery shopping, Attend meeting (if optional)

---

### 3. Task Blocker Solver

## QUICK REFERENCE GUIDE

### How to Use This Prompt Library

**The 3-Minute Quick Start:**
1. Find a prompt that matches your current challenge
2. Copy the exact prompt text
3. Fill in [BRACKETS] with your specific information
4. Paste into ChatGPT and hit enter
5. Use the output immediately

### Best Practices:
- **Be specific** - The more detail you provide in brackets, the more useful the output
- **Go deep** - Don't settle for first answer; ask follow-up questions
- **Iterate** - Run the same prompt weekly or monthly to track evolution
- **Personalize** - Modify prompts to match your language and style
- **Experiment** - Try different prompts for the same problem

### Categories Cheat Sheet:
- **Overwhelmed with tasks?** → Category 1 (Task Management)
- **Drowning in meetings?** → Category 2 (Time Management)
- **Need better systems?** → Category 3 (Productivity Systems)
