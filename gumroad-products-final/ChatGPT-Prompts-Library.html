Category 1: Task Management & TODO (Prompts 1-10)
1Smart Task Breakdown Prompt
Prompt
"I have this task: [TASK]. Break it down into 5-7 specific, actionable subtasks that I can realistically complete today. For each subtask, estimate the time needed and difficulty level (easy/medium/hard). Format as a numbered checklist I can use immediately."
Use Case
Breaking large projects into manageable daily chunks
Example
Input: Write a quarterly business report | Output: 1. Gather Q4 sales data (30 min-Easy) | 2. Compile customer feedback (45 min-Medium) | 3. Create charts/visuals (60 min-Medium)
Pro Tip
Be specific about the main task. The more detail you provide, the better the breakdown will be.
2Priority Matrix Generator
Prompt
"I have these 8 tasks to do this week: [LIST TASKS]. Use the Eisenhower Matrix (urgent/important) to categorize them. Show me which I should do first, which I should schedule, which I should delegate, and which I should eliminate. Explain your reasoning for each."
Use Case
Deciding what to focus on when overwhelmed with competing priorities
Example
Lists 8 tasks across four quadrants: DO FIRST (urgent & important), SCHEDULE (important but not urgent), DELEGATE (urgent but not important), ELIMINATE (neither)
Pro Tip
Most productive people spend 80% of time in the "important but not urgent" quadrant.
3Task Blocker Solver
Prompt
"I'm stuck on this task: [DESCRIBE BLOCKER]. What's preventing me from moving forward? Give me 3 specific, actionable steps to overcome this blocker right now."
Use Case
Getting unstuck when you don't know what to do or why you're procrastinating
Example
Input: Need to write client proposal but keep procrastinating | Output: Step 1: Write outline (5 min - reduces perfectionism) | Step 2: 25-min timer, write intro (creates momentum) | Step 3: Read aloud, fix gaps (builds confidence)
Pro Tip
Procrastination often masks fear of perfectionism or unclear expectations. Identify the real blocker first.
4Daily Planning Template Generator
Prompt
"Create a detailed plan for my tomorrow using this info: [SCHEDULE OVERVIEW, MAIN GOALS]. Include: time blocks, break times, backup/flex time, success metrics, one success definition."
Use Case
Planning the perfect productive day with all commitments and energy management
Example
Input: 8am meeting, finish report, 3 client calls, exercise | Output: Hour-by-hour schedule with time blocks, breaks at 10am and noon, success metric defined
Pro Tip
Include 30% buffer time. Most plans fail because they don't account for interruptions.
5Recurring Task Automation Checker
Prompt
"I do these tasks repeatedly: [LIST RECURRING TASKS]. For each one, tell me: Can it be automated? What tool? How much time saved per month? Implementation steps."
Use Case
Finding easy automations to free up hours of time every month
Example
Weekly emails via IFTTT (5 hrs/mo) | Meeting notes via Otter.ai (8 hrs/mo) | Invoicing via Stripe (4 hrs/mo) | Reminders via calendar automation (2 hrs/mo) = 19 hours recovered
Pro Tip
Focus on tasks done 2+ times per month. Even small automations compound to huge time savings.
6Parallel Task Organizer
Prompt
"I need to juggle these [NUMBER] tasks: [LIST THEM]. Which can I do in parallel? Which must be sequential? Create a workflow showing dependencies."
Use Case
Managing multiple projects at once and understanding what can overlap
Example
Creates sequential flowchart showing which tasks have dependencies, which can overlap, critical path, timeline with overlaps to compress schedule
Pro Tip
Critical path analysis saves time. Many projects can be compressed 20-30% by better parallelization.
7Task Estimation Calibrator
Prompt
"I estimated these tasks would take [TIMES] but they usually take [ACTUAL TIMES]. Show me the pattern. Give me a personal formula to estimate more accurately."
Use Case
Getting better at time estimates by understanding your personal bias
Example
Est 1hr→Actual 2hrs, Est 2hrs→Actual 3.5hrs | Output: Shows 1.5-1.75x multiplier pattern. Suggests 15% buffer and time-boxing technique.
Pro Tip
Track estimates vs actuals for 2+ weeks first. Most people consistently underestimate by 1.5-2x.
8Context Switching Reducer
Prompt
"I context switch between [LIST TASKS/APPS]. Create a schedule that: batches similar tasks, minimizes app switching, protects deep work time, handles urgent items."
Use Case
Eliminating the #1 productivity killer - constant context switching
Example
Before: Jump between emails, Slack, coding, meetings | After: Batch emails 3x daily, Slack in 2 blocks, 2hr coding blocks, meetings in afternoon
Pro Tip
Each context switch costs 15-25 minutes of refocusing time. Aim for 2-3 switches per day max.
9Completed Tasks Analyzer
Prompt
"Over the past [WEEK/MONTH], I completed these tasks: [LIST]. What patterns do you see? What energized vs drained me? What should I do more/less of?"
Use Case
Learning from what actually happened to improve future planning
Example
Input: List of 20 completed tasks from past week | Output: Energy patterns, high-impact vs low-impact work, specific recommendations for next week
Pro Tip
Include small tasks too. Most people are surprised what actually energizes vs drains them.
10Task Refusal Decision Maker
Prompt
"Someone asked me to do this: [TASK DESCRIPTION]. Should I say yes or no? Evaluate: strategic fit, energy/time cost, opportunity cost, real urgency."
Use Case
Protecting your time and energy with confidence and professional language
Example
Input: Manager asked to lead new initiative on top of current work | Output: Decision framework with fit analysis, cost breakdown, alternatives, confidence level with scripts
Pro Tip
Use this when you're unsure about saying no. The structured framework helps you decide and respond professionally.

How To Use This Library

The 3-Minute Quick Start

  1. Find a prompt matching your current challenge
  2. Copy the exact prompt text
  3. Fill in [BRACKETS] with your specific information
  4. Paste into ChatGPT and hit enter
  5. Use the output immediately

Best Practices

  • Be Specific — More detail yields better output
  • Go Deep — Ask follow-up questions for insights
  • Iterate — Run the same prompt monthly to track evolution
  • Personalize — Modify to match your language and style
  • Experiment — Try different prompts for the same problem
